+ Calificaciones

Proyecto en C++ que permite calcular y almacenar las calificaciones de varios estudiantes.  
Cada estudiante puede tener la cantidad de materias que el usuario desee, y cada materia tiene 3 parciales.

El programa calcula:
- Promedio por materia  
- Promedio final del estudiante  
- Estado: **Aprobado** (>= 6.0) o **Reprobado** (< 6.0)  
- Guarda los datos de cada estudiante ingresado  
- Al finalizar, genera un **reporte general** con todos los estudiantes registrados



+ Características principales

- Permite ingresar **varios estudiantes**
- Permite elegir **cuántas materias** tendrá cada estudiante
- Cada materia tiene **3 calificaciones**
- Calcula:
  - Promedio por materia
  - Promedio final del estudiante
  - Estado del estudiante según el promedio final
- Al terminar, muestra un **reporte completo** con:
  - Nombres ingresados de los estudiantes
  - Promedios por materia
  - Promedio final
  - Estado de Aprobado/Reprobado


* Compilación e instalación
Para poder ejecutar el código, es necesario contar con un compilador como lo puede ser dev c++, apache NetBeans o algún compilador en linea son software abierto.

Requisitos
- Compilador C++
- Instalación correcta de dev c++ o Apache NetBeans Según sea el caso.

¿Cómo usar el programa?

Ejecuta el programa.

Ingresa el nombre del estudiante.

Ingresa cuántas materias llevará.

Para cada materia, ingresa 3 parciales (valores entre 0 y 10).

El programa mostrará:

Promedio por cada materia

Promedio final

Estado de aprobado o reprobado

Al final, podrás elegir:

1  Registrar otro estudiante

2  Finalizar y mostrar el reporte completo

Si ingresas cualquier otro número, también se finaliza mostrando el reporte.

=============================================
